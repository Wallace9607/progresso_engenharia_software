Projeto de Balanço em Árvore

Levantamento de Requisitos

Atividade da disciplina de Princípios de Engenharia de Software sobre especificação em linguagem natural e técnicas de elicitação.

Disciplina: Princípios de Engenharia de Software

Aluno: Wallace Alves de Freitas

Contexto do Projeto

Uma dona de chácara solicitou a construção de um balanço em uma árvore localizada em seu terreno. O objetivo é criar um espaço de lazer para crianças que visitam o local.

  • Local: chácara com área arborizada
  • Objetivo: criar um brinquedo seguro para crianças
  • Instalação: balanço preso a um galho resistente

Especificação em Linguagem Natural

Durante a conversa com a cliente, foram levantadas as seguintes necessidades para o balanço na árvore:

  • O balanço deve ser instalado em um galho forte da árvore.
  • Deve utilizar cordas resistentes para suportar o peso dos usuários.
  • O assento deve ser firme e confortável.
  • A altura do balanço deve permitir que crianças subam com facilidade.
  • Os materiais devem ser resistentes ao sol e à chuva.

Principais Requisitos do Balanço

Segurança

O balanço deve suportar o peso dos usuários com cordas e fixação resistentes.

Estrutura

Deve ser instalado em um galho forte e estável da árvore.

Conforto

O assento deve ser firme e confortável para as crianças.

Durabilidade

Os materiais devem resistir ao sol, chuva e uso frequente.

Técnicas de Elicitação de Requisitos

Entrevista

Consiste em conversar diretamente com a dona da chácara para entender suas necessidades, expectativas e requisitos para o balanço.

  • Identificar necessidades
  • Entender expectativas
  • Levantar requisitos iniciais

Observação

Consiste em observar balanços semelhantes em parques ou áreas de lazer, para identificar boas práticas de construção e segurança.

  • Analisar uso real
  • Identificar melhorias
  • Validar requisitos

Conclusão

  • A elicitação de requisitos é essencial para compreender as necessidades do usuário.

  • Com base nesses dados, é possível definir requisitos claros para o projeto.
